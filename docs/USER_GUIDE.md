# Athar — Guide utilisateur

> **Athar** (أثر) — Traces & vestiges
> L'atelier d'enquête historique sur mobile

---

## Démarrage rapide

### Première utilisation

1. Ouvrez l'application
2. L'écran d'accueil affiche un message de bienvenue
3. Commencez par créer un **dossier de recherche** (onglet Dossiers → Nouveau dossier)
4. Ajoutez vos premières **entités** (personnes, lieux, événements) via l'onglet Ajouter

### Navigation

L'application a **5 onglets principaux** :

| Onglet | Icône | Usage |
|---|---|---|
| **Accueil** | 🏠 | Tableau de bord, stats, actions rapides, entités récentes |
| **Dossiers** | 📁 | Vos dossiers de recherche |
| **Ajouter** | ➕ | Hub central pour tout créer |
| **Frise** | 🕐 | Chronologie de vos données |
| **Chercher** | 🔍 | Recherche dans toutes les entités |

---

## Les dossiers de recherche

Un dossier est l'unité centrale de votre travail. Il regroupe tout ce qui concerne une enquête.

### Créer un dossier

1. Onglet **Dossiers** → **Nouveau dossier**
2. Remplissez :
   - **Titre** : ex. "Migration kabyle vers la France, 1900-1960"
   - **Question de recherche** : la problématique qui guide votre enquête
   - **Période** : dates de début et fin
   - **Aire géographique** : régions couvertes
   - **Statut** : Actif, En pause, Terminé, Archivé
3. Sauvegardez

### Dans un dossier, vous pouvez :
- Voir vos **sources**, **hypothèses** et **notes** liées
- Ajouter rapidement une source, hypothèse ou note
- **Exporter** le dossier en PDF ou HTML

## Les entités historiques

### 4 types d'entités

| Type | Exemples |
|---|---|
| **Personne** | Ahmed Ben Mostefa, Yamina Ait Kaci |
| **Groupe** | Famille Ben Mostefa, FLN, Association des Kabyles de Paris |
| **Lieu** | Tizi Ouzou, Boulogne-Billancourt, Camp de Rivesaltes |
| **Événement** | Migration vers la France (1925), Toussaint Rouge (1954) |

### Créer une entité

Onglet **Ajouter** → section **Entités** → choisir le type

Chaque entité a :
- Un nom (obligatoire)
- Un type spécifique (type de lieu, type d'événement, etc.)
- Un résumé / description
- Des tags pour le classement
- Des dates (quand applicable)

### Fiche entité

Tapez sur une entité pour voir sa fiche :
- **En-tête** : badge coloré, nom, type
- **Champs** : dates, variantes de nom, description
- **Tags** : étiquettes ambrées
- **Relations** : liens vers d'autres entités avec niveau de confiance
- **Actions** : ajouter une relation, voir le réseau, supprimer

---

## Les relations

Les relations connectent deux entités entre elles.

### Créer une relation

1. Depuis une fiche entité → **Relation**
2. Ou depuis Ajouter → Recherche → **Relation**
3. Sélectionnez :
   - **Entité source** et **entité cible**
   - **Type** : parenté, alliance, voisinage, appartenance, etc.
   - **Libellé** : ex. "père", "président", "voisin"
   - **Niveau de confiance** : confirmé, probable, incertain, contesté

### Types de relations disponibles
Parenté, Alliance, Voisinage, Appartenance, Fonction, Institutionnel, Participation, Dépendance, Transmission, Économique, Religieux, Militaire, Politique, Social, Autre

## Les sources

### Créer une source

Ajouter → **Nouvelle source**

Une source complète contient :
- **Titre** et **type** (primaire, secondaire, témoignage, archive privée, note interne)
- **Auteur / Producteur**
- **Provenance** et **lieu de conservation**
- **Cote** et **fonds** d'archives
- **Langue**
- **Fiabilité** : haute, moyenne, basse, inconnue
- **Biais / réserves** : limites connues de la source
- **Résumé analytique**

### Extraits

Dans une fiche source, vous pouvez ajouter des **extraits** — des passages clés avec :
- Le texte sélectionné
- La localisation (page, folio, timestamp)
- Une **classification** :
  - 🟢 **Preuve** : établit un fait
  - 🟠 **Indice** : suggère sans prouver
  - ⚪ **Contexte** : éclaire le cadre
  - 🟡 **Contradiction** : contredit une autre source
  - 🔴 **Doute** : soulève une question

## Capture terrain

### Photo d'archives

1. Ajouter → **Photo d'archive**
2. Prenez la photo ou importez depuis la galerie
3. Ajoutez titre et description
4. **Liez à des entités** : dépliez le sélecteur pour associer la photo à des personnes, lieux, etc.
5. Sauvegardez

### Témoignage vocal

1. Ajouter → **Témoignage vocal**
2. Appuyez sur le bouton micro pour enregistrer
3. Pause / reprise / arrêt possibles
4. Remplissez titre et nom du témoin
5. Liez à des entités
6. Sauvegardez

### Fiches témoins

Ajouter → Terrain → **Témoin**

Documentez vos témoins avec :
- Identité, naissance, localisation
- Relation au sujet d'étude
- **Consentement** : obtenu, en attente, restreint, refusé
- **Sensibilité** : public, normal, sensible, confidentiel

### Missions terrain

Ajouter → Terrain → **Mission**

Planifiez vos déplacements :
- Objectifs, dates, lieu
- Personnes à rencontrer
- Lieux à visiter
- Archives à consulter
- Matériel nécessaire

## Hypothèses

### Créer une hypothèse

Ajouter → Recherche → **Hypothèse**

Une hypothèse a :
- Un **titre** : l'affirmation proposée
- Une **description** : l'argumentaire développé
- Un **statut** : Brouillon → Ouverte → Argumentée → Validée provisoirement / Rejetée
- Un **niveau de confiance** : confirmé, probable, incertain, contesté, abandonné

### Bonnes pratiques
- Formulez l'hypothèse comme une proposition testable
- Décrivez les arguments pour ET contre dans la description
- Mettez à jour le statut au fur et à mesure de vos découvertes

---

## Contradictions

Ajouter → Recherche → **Contradiction**

Quand deux sources se contredisent, documentez-le :
- Titre descriptif
- Description des divergences
- Statuts : ouverte → résolue / reconnue

---

## Chaînes de preuve (V3)

Ajouter → Recherche → **Preuve**

Reliez une affirmation à ses appuis documentaires :
1. Formulez l'**affirmation à démontrer**
2. Évaluez son **statut** : non vérifié → étayé → contesté → réfuté
3. Ajoutez des **maillons** : chaque source, extrait ou témoignage qui soutient ou conteste

---

## Prosopographie (V3)

Ajouter → Recherche → **Cohorte**

Étudiez un groupe d'individus par champs comparables :
1. Définissez les **critères** de la cohorte
2. Ajoutez des personnes avec des champs structurés :
   - Origine sociale, niveau d'éducation, occupation
   - Affiliation politique, service militaire
   - Date et destination de migration
   - Statut familial, événements notables

## Chronologie

L'onglet **Frise** affiche tous vos éléments datés :
- 🟢 Naissances
- ⚫ Décès
- 🟠 Événements
- 🟤 Sources
- 🟣 Témoignages

### Filtres
Utilisez les pills en haut pour filtrer par couche : Tout, Faits, Sources, Témoignages.

---

## Graphe relationnel

Depuis l'accueil → **Réseau** ou depuis une fiche entité → **Réseau**

Le graphe affiche :
- Nœuds colorés par type d'entité
- Liens entre entités connectées
- Tap sur un nœud = sélection + détail
- Filtrage par type d'entité
- Zoom et scroll

---

## Recherche

L'onglet **Chercher** permet de trouver n'importe quelle entité par :
- Nom
- Tags
- Variantes de nom (aliases)

Tapez au moins 2 caractères pour lancer la recherche.

---

## Export

Depuis un dossier → **Exporter ce dossier**

Formats disponibles :
- **PDF** : dossier complet prêt à imprimer
- **HTML** : page web consultable hors-ligne

L'export inclut :
- Métadonnées du dossier
- Tableau des personnes, événements, lieux, groupes
- Relations avec niveaux de confiance
- Sources avec extraits en citations
- Hypothèses
- Notes de recherche

---

## Bibliographie

Ajouter → Terrain → **Biblio**

Types d'entrées : livre, article, thèse, chapitre, rapport, site web, guide d'archives.

Champs : auteurs, année, éditeur/revue, pages, URL, ISBN, résumé analytique.

## Conseils de workflow

### Pour une enquête complète

1. **Créez un dossier** avec votre question de recherche
2. **Ajoutez vos sources** au fur et à mesure des découvertes
3. **Extrayez les passages clés** avec leur classification
4. **Créez les entités** mentionnées dans vos sources
5. **Reliez tout** : entités entre elles, sources aux entités, preuves aux photos
6. **Formulez des hypothèses** quand vous avez assez d'éléments
7. **Documentez les contradictions** entre sources
8. **Construisez les chaînes de preuve** pour vos affirmations centrales
9. **Exportez** votre dossier pour partage ou publication

### Pour le terrain

1. **Préparez une mission** avec objectifs et checklist
2. **Créez les fiches témoins** avec gestion du consentement
3. **Enregistrez les entretiens** avec liaison aux entités
4. **Photographiez les archives** avec métadonnées
5. **Prenez des notes terrain** liées au dossier
6. De retour, **croisez** avec les archives écrites
